
# Introduction

This is a simple CORS proxy server implemented in Node.js using Express. It allows you to bypass CORS restrictions by forwarding requests to the target URL and returning the response back to the client.

# Usage


1. Clone this project:

   ```bash
   git clone https://github.com/ZXS66/cors-proxy.git
   # git clone git@github.com:ZXS66/cors-proxy
   ```
2. Install dependencies:

   ```bash
   pnpm install
   ```
3. Configure environment variables (if needed):
   
   Copy the `.env.example` file to `.env` and modify the settings as required.
   
   ```bash
   cp .env.example .env
   ```
4. Start the server:
   ```bash
   ./run.sh
   ```
